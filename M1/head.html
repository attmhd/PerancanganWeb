Tag < head>: Bagian ini berisi metadata tentang dokumen, yang tidak langsung ditampilkan di halaman web tetapi digunakan oleh mesin pencari, browser, dan perangkat lainnya.

Tag < meta charset="UTF-8">: Tag meta ini mendefinisikan pengkodean karakter yang digunakan dalam dokumen. UTF-8 adalah pengkodean yang banyak digunakan yang mendukung berbagai karakter, membuatnya cocok untuk sebagian besar konten web.

Tag < meta name="viewport" content="width=device-width, initial-scale=1.0">: Tag meta ini mengonfigurasi bagaimana halaman web akan ditampilkan di berbagai perangkat. Ini menginstruksikan browser untuk mengatur lebar awal viewport (area halaman yang terlihat) ke lebar perangkat dan menggunakan skala awal 1.0, memastikan bahwa konten tidak diperbesar atau diperkecil secara default. Ini memungkinkan desain web responsif, di mana tata letak beradaptasi dengan berbagai ukuran layar.
